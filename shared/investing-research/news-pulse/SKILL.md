# 股价异动快速归因（news-pulse）

**不是深度投研，是10-15分钟的快速归因。**

当持仓股大涨/大跌时，快速搞清"发生了什么"，避免陷入小作文焦虑或盲目止损。

---

## 核心理念

**归因优先于罗列**：不是把所有新闻列一遍，而是判断"哪个事件配得上这次股价异动"。

**强制性质判断**：必须给出以下四种判断之一：
- **价值事件**：基本面发生了真实变化
- **情绪波动**：市场情绪导致的短期波动
- **真因不明**：找不到合理解释（可能存在内幕抢跑）
- **混合型**：多个因素共同作用

---

## 分析流程

### 第一步：4维并行侦察

同时从4个维度搜索信息：

| 维度 | 搜索内容 | 目的 |
|------|----------|------|
| **公司事件** | 公司公告、财报、高管变动 | 看公司自身有没有变化 |
| **监管政策** | 行业政策、监管动态 | 看外部环境有没有变化 |
| **行业对手** | 竞争对手动态、行业新闻 | 看竞争格局有没有变化 |
| **市场情绪** | 卖方研报、大V观点、资金流向 | 看市场情绪有没有变化 |

**数据获取**：
- `mcp_tavily_mcp_tavily_search` 搜索"{股票名} {日期} 新闻"
- `mcp_tavily_mcp_tavily_search` 搜索"{行业} 政策 最新"
- `mcp_tavily_mcp_tavily_search` 搜索"{股票名} 资金流 龙虎榜"

**扩展场景：大盘异动归因**
当大盘整体下跌/上涨时，搜索维度扩展为：
- `mcp_tavily_mcp_tavily_search` 搜索"A股 {日期} 大盘 下跌 原因"
- `mcp_tavily_mcp_tavily_search` 搜索"全球股市 {日期} 联动"
- `mcp_tavily_mcp_tavily_search` 搜索"美联储 {日期} 政策"
- 关注美联储政策、地缘政治、宏观经济数据等宏观因素

**大盘异动归因模板差异**：
- 异动概况：改为三大指数数据（上证/深证/创业板）
- 4维侦察：改为宏观因素（美联储政策、地缘政治、经济数据、资金流向）
- 异动归因表：改为宏观因素贡献度
- 性质判断：同样适用（价值事件/情绪波动/真因不明/混合型）

### 第二步：异动归因表

列出所有可能的解释，并估算贡献度：

```
异动归因表：

| 候选解释 | 估算贡献 | 置信度 |
|---------|---------|--------|
| [事件1] | XX% | 高/中/低 |
| [事件2] | XX% | 高/中/低 |
| [事件3] | XX% | 高/中/低 |
| 基本面恶化 | XX% | 高/中/低 |
```

**关键判断**：
- 如果"基本面恶化"贡献>50% → 价值事件，需要重新审视投资论文
- 如果"基本面恶化"贡献=0% → 情绪波动，可能是加仓机会
- 如果找不到合理解释 → 真因不明，可能存在内幕抢跑

### 第三步：性质判断

必须给出以下四种判断之一：

| 性质 | 特征 | 行动建议 |
|------|------|----------|
| **价值事件** | 基本面发生真实变化 | 重新审视投资论文，考虑止损/加仓 |
| **情绪波动** | 市场情绪导致的短期波动 | 持有/加仓（如果基本面没变） |
| **真因不明** | 找不到合理解释 | 警惕，可能存在内幕信息 |
| **混合型** | 多个因素共同作用 | 具体分析各因素权重 |

### 第四步：行动建议

根据性质判断，给出明确的行动建议：

```
行动建议：

1. [ ] 触发深度研究：是/否
2. [ ] 需要重审论文：是/否
3. [ ] 考虑加仓：是/否
4. [ ] 考虑止损：是/否
5. [ ] 仅观察：是/否
```

## 注意事项

1. **速度优先**：10-15分钟内完成，不要追求完美
2. **归因优先**：不是列新闻，是判断"哪个事件配得上这次异动"
3. **强制判断**：必须给出性质判断，不能模棱两可
4. **行动导向**：必须给出明确的行动建议
