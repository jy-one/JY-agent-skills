# CPO（共封装光学）行业分析

## 行业概况
- **定义**：Co-Packaged Optics，将光引擎与交换芯片共同封装在同一基板上，取代传统可插拔光模块
- **市场规模**：2025年约1亿美元（导入期），2026年规模化商用元年（12-19亿美元），2035年预计超190亿美元
- **发展阶段**：导入期 → 成长期过渡
- **技术路线**：硅光技术为主流

## 产业链
```
硅光芯片 → 光引擎封装 → 交换芯片(ASIC) → 系统集成 → 云厂商部署
```

## 投资逻辑与风险
- **核心逻辑**：2025导入期，2026规模化商用，AI算力对带宽需求是刚性的
- **技术路线风险**：NPO（近封装光学）可能分流CPO需求
- **商业化风险**：成本高，大规模部署可能延后
- **估值风险**：A股CPO标的普遍偏贵
