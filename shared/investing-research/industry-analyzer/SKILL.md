# 行业分析框架

配合 `stock-analysis-base` 使用。本skill专注**行业层面**分析，自上而下。

## 分析顺序

```
行业概况 → 产业链拆解 → 竞争格局 → 技术路线 → 行业周期 → 政策环境 → 投资机会
```

---

## 一、行业概况

**核心问题**：这个行业是干什么的？有多大？在增长吗？

### 1.1 行业定义与分类

- 行业定义：这个行业提供什么产品/服务
- 细分领域：行业内有哪些子赛道
- 产业链位置：处于整个产业链的哪个环节

### 1.2 市场规模

| 指标 | 看什么 | 数据来源 |
|------|--------|----------|
| 全球市场规模 | 行业整体体量 | Prismark、Yole、IDC等 |
| 国内市场规模 | 国产替代空间 | 行业协会、券商研报 |
| 增速 | 行业景气度 | >10%算高增长 |
| 渗透率 | 发展阶段 | <30%还有空间 |

### 1.3 发展阶段

| 阶段 | 特征 | 投资逻辑 |
|------|------|----------|
| 导入期 | 技术刚成熟、市场小、增速快 | 高风险高回报 |
| 成长期 | 需求爆发、产能扩张、玩家增多 | 买龙头 |
| 成熟期 | 增速放缓、格局稳定、利润见顶 | 买龙头、吃分红 |
| 衰退期 | 需求下滑、技术被替代 | 回避 |

## 二、产业链拆解

**核心问题**：上中下游分别是谁？谁赚的钱最多？

### 2.1 产业链结构

```
上游（原材料/设备）→ 中游（制造/加工）→ 下游（应用/终端）
```

### 2.2 各环节分析

| 环节 | 分析要点 | 关键问题 |
|------|----------|----------|
| 上游 | 供给集中度、技术壁垒、价格走势 | 是否卡脖子？ |
| 中游 | 产能利用率、竞争格局、毛利率 | 谁是龙头？ |
| 下游 | 需求增速、客户集中度、议价能力 | 需求是否可持续？ |

### 2.3 价值分配

**关键判断**：
- 毛利率最高的环节 = 价值链核心
- 壁垒最高的环节 = 最难被替代
- 话语权最强的环节 = 定价权在手

## 三、竞争格局

**核心问题**：行业集中度如何？谁是龙头？进入壁垒有多高？

### 3.1 集中度分析

| 指标 | 含义 | 参考值 |
|------|------|--------|
| CR3 | 前三家市占率 | >50%=高集中度 |
| CR5 | 前五家市占率 | >70%=寡头垄断 |
| CR10 | 前十家市占率 | >80%=高度集中 |

### 3.2 竞争格局类型

| 类型 | 特征 | 投资策略 |
|------|------|----------|
| 完全垄断 | 一家独大，市占率>60% | 买垄断者 |
| 寡头垄断 | CR3>70%，定价权强 | 买龙头 |
| 双龙头 | 两家份额>50% | 二选一 |
| 一超多强 | 一家领先，多家追赶 | 买龙头或等第二名突围 |
| 完全竞争 | 分散、无壁垒 | 回避或做波段 |

### 3.3 进入壁垒

| 壁垒类型 | 高壁垒特征 |
|---------|-----------|
| 技术壁垒 | 认证周期长、研发投入大 |
| 资金壁垒 | 投资额大、回收期长 |
| 客户壁垒 | 认证周期长、客户粘性强 |
| 规模壁垒 | 大厂成本显著低于小厂 |
| 政策壁垒 | 需要特殊资质 |

### 4.1 技术路线图

| 技术 | 成熟度 | 应用场景 | 代表公司 |
|------|--------|----------|----------|
| 当前主流 | 成熟 | | |
| 下一代 | 发展中 | | |
| 未来方向 | 研发中 | | |

### 4.2 国产替代进度

| 环节 | 进口依赖度 | 国产替代进度 | 突破公司 |
|------|-----------|-------------|----------|
| | | 刚起步/加速期/成熟期 | |

**投资逻辑**：
- 刚起步（<20%）：弹性最大、但风险也大
- 加速期（20-50%）：确定性较高
- 成熟期（>50%）：看龙头份额提升

## 五、行业周期

**核心问题**：当前处于周期哪个位置？驱动因素是什么？

### 5.1 周期位置判断

```
底部 → 上行 → 顶部 → 下行 → 底部
```

| 周期位置 | 特征 | 策略 |
|---------|------|------|
| 底部 | 产能过剩、价格战、厂商亏损 | 布局龙头，等待出清 |
| 上行 | 需求回暖、涨价、产能紧张 | 重仓持有 |
| 顶部 | 全行业暴利、疯狂扩产 | 逐步减仓 |
| 下行 | 需求下滑、库存累积、价格下跌 | 回避或做空 |

### 5.2 周期判断方法

| 指标 | 底部信号 | 顶部信号 |
|------|----------|----------|
| 产品价格 | 跌至成本线、开始涨价 | 暴涨、创新高 |
| 产能利用率 | <60% | >90% |
| 库存水平 | 低位、去库存 | 高位、累库存 |
| 厂商行为 | 亏损、减产、退出 | 暴利、疯狂扩产 |
| 新进入者 | 无人进入 | 大量跨界玩家涌入 |
| 资本市场 | 无人关注、估值低 | 热炒、估值高 |

### 5.3 驱动因素

| 因素 | 类型 | 影响 |
|------|------|------|
| 需求端 | 新能源车、AI、5G等 | 拉动需求 |
| 供给端 | 产能扩张、技术突破 | 影响供给 |
| 政策端 | 补贴、准入、环保 | 加速或抑制 |
| 成本端 | 原材料价格、能源成本 | 影响利润 |
