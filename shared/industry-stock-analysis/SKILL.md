# PART 1：行业分析框架

## 分析顺序

```
行业概况 → 产业链拆解 → 竞争格局 → 技术路线 → 行业周期 → 政策环境 → 投资机会
```

## 一、行业概况

**核心问题**：这个行业是干什么的？有多大？在增长吗？

### 1.1 行业定义与分类

| 维度 | 内容 |
|------|------|
| 行业定义 | 这个行业提供什么产品/服务 |
| 细分领域 | 行业内有哪些子赛道 |
| 产业链位置 | 处于整个产业链的哪个环节 |

### 1.2 市场规模

| 指标 | 看什么 | 数据来源 |
|------|--------|----------|
| 全球市场规模 | 行业整体体量 | Prismark、Yole、IDC等 |
| 国内市场规模 | 国产替代空间 | 行业协会、券商研报 |
| 增速 | 行业景气度 | >10%算高增长 |
| 渗透率 | 发展阶段 | <30%还有空间 |

### 1.3 发展阶段

| 阶段 | 特征 | 投资逻辑 |
|------|------|----------|
| 导入期 | 技术刚成熟、市场小、增速快 | 高风险高回报 |
| 成长期 | 需求爆发、产能扩张、玩家增多 | 买龙头 |
| 成熟期 | 增速放缓、格局稳定、利润见顶 | 买龙头、吃分红 |
| 衰退期 | 需求下滑、技术被替代 | 回避 |

## 二、产业链拆解

**核心问题**：上中下游分别是谁？谁赚的钱最多？

### 2.1 产业链结构

```
上游（原材料/设备）→ 中游（制造/加工）→ 下游（应用/终端）
```

### 2.2 各环节分析

| 环节 | 分析要点 | 关键问题 |
|------|----------|----------|
| 上游 | 供给集中度、技术壁垒、价格走势 | 是否卡脖子？ |
| 中游 | 产能利用率、竞争格局、毛利率 | 谁是龙头？ |
| 下游 | 需求增速、客户集中度、议价能力 | 需求是否可持续？ |

### 2.3 价值分配

| 环节 | 毛利率 | 壁垒 | 话语权 |
|------|--------|------|--------|
| 上游 | 高/中/低 | 高/中/低 | 强/中/弱 |
| 中游 | 高/中/低 | 高/中/低 | 强/中/弱 |
| 下游 | 高/中/低 | 高/中/低 | 强/中/弱 |

**关键判断**：
- 毛利率最高的环节 = 价值链核心
- 壁垒最高的环节 = 最难被替代
- 话语权最强的环节 = 定价权在手

## 三、竞争格局

**核心问题**：行业集中度如何？谁是龙头？进入壁垒有多高？

### 3.1 集中度分析

| 指标 | 含义 | 参考值 |
|------|------|--------|
| CR3 | 前三家市占率 | >50%=高集中度 |
| CR5 | 前五家市占率 | >70%=寡头垄断 |
| CR10 | 前十家市占率 | >80%=高度集中 |
| HHI | 赫芬达尔指数 | >2500=高集中度 |

### 3.2 竞争格局类型

| 类型 | 特征 | 投资策略 |
|------|------|----------|
| 完全垄断 | 一家独大，市占率>60% | 买垄断者 |
| 寡头垄断 | CR3>70%，定价权强 | 买龙头 |
| 双龙头 | 两家份额>50% | 二选一 |
| 一超多强 | 一家领先，多家追赶 | 买龙头或等第二名突围 |
| 完全竞争 | 分散、无壁垒 | 回避或做波段 |

### 3.3 进入壁垒

| 壁垒类型 | 内容 | 高壁垒特征 |
|---------|------|-----------|
| 技术壁垒 | 专利、Know-how、工艺 | 认证周期长、研发投入大 |
| 资金壁垒 | 建厂、设备、研发投入 | 投资额大、回收期长 |
| 客户壁垒 | 认证、粘性、切换成本 | 认证周期长、客户粘性强 |
| 规模壁垒 | 规模效应、成本优势 | 大厂成本显著低于小厂 |
| 政策壁垒 | 牌照、资质、准入 | 需要特殊资质 |

## 五、行业周期

**核心问题**：当前处于周期哪个位置？驱动因素是什么？未来怎么走？

### 5.1 周期位置判断

```
底部 → 上行 → 顶部 → 下行 → 底部
```

| 周期位置 | 特征 | 策略 |
|---------|------|------|
| 底部 | 产能过剩、价格战、厂商亏损 | 布局龙头，等待出清 |
| 上行 | 需求回暖、涨价、产能紧张 | 重仓持有 |
| 顶部 | 全行业暴利、疯狂扩产 | 逐步减仓 |
| 下行 | 需求下滑、库存累积、价格下跌 | 回避或做空 |

### 5.2 周期判断方法

| 指标 | 底部信号 | 顶部信号 |
|------|----------|----------|
| 产品价格 | 跌至成本线、开始涨价 | 暴涨、创新高 |
| 产能利用率 | <60% | >90% |
| 库存水平 | 低位、去库存 | 高位、累库存 |
| 厂商行为 | 亏损、减产、退出 | 暴利、疯狂扩产 |
| 新进入者 | 无人进入 | 大量跨界玩家涌入 |
| 资本市场 | 无人关注、估值低 | 热炒、估值高 |

### 5.3 驱动因素

| 因素 | 类型 | 影响 |
|------|------|------|
| 需求端 | 新能源车、AI、5G等 | 拉动需求 |
| 供给端 | 产能扩张、技术突破 | 影响供给 |
| 政策端 | 补贴、准入、环保 | 加速或抑制 |
| 成本端 | 原材料价格、能源成本 | 影响利润 |

# PART 2：个股分析框架

## 分析顺序

```
行业格局 → 公司定位 → 管理层分析 → 财务验证 → 股东博弈 → 可比对比 → 产业链验证 → 结论
```

## 一、行业格局分析（自上而下）

**核心问题**：这个行业有没有技术壁垒？国产替代到哪一步了？当前处于周期哪个位置？

### 1.1 产业链拆解

| 维度 | 具体内容 | 分析要点 |
|-----|---------|---------|
| 上游 | 原材料、设备、零部件 | 供给是否受限、价格走势 |
| 中游 | 制造、加工、组装 | 技术壁垒、产能利用率 |
| 下游 | 终端应用、品牌商 | 需求增速、集中度 |

### 1.2 技术壁垒评估

| 壁垒等级 | 特征 | 典型行业 |
|---------|------|---------|
| 极高 | 技术密集、资金密集、认证周期长 | 半导体设备、IC载板 |
| 高 | 需要长期积累、客户粘性强 | 高端覆铜板、精密零部件 |
| 中 | 有一定技术门槛、可追赶 | PCB、锂电池材料 |
| 低 | 同质化竞争、拼成本 | 普通钢材、基础化工 |

### 1.3 国产替代进度

| 进度 | 特征 | 投资逻辑 |
|-----|------|---------|
| 刚起步（<20%） | 进口依赖度高、政策扶持 | 弹性最大、但风险也大 |
| 加速期（20-50%） | 龙头已突破、放量在即 | 确定性较高 |
| 成熟期（>50%） | 国产已主导、竞争加剧 | 看龙头份额提升 |

### 1.4 竞争格局

| 格局 | 特征 | 投资策略 |
|-----|------|---------|
| 寡头垄断 | CR3>70%，定价权强 | 买龙头 |
| 双龙头 | 两家份额>50%，稳定 | 二选一 |
| 一超多强 | 一家独大，其他追赶 | 买龙头或等第二名突围 |
| 完全竞争 | 分散、无壁垒 | 回避或做波段 |

### 1.5 行业周期位置

```
底部 → 上行 → 顶部 → 下行 → 底部
```

| 周期位置 | 特征 | 策略 |
|---------|------|------|
| 底部 | 产能过剩、价格战、厂商亏损 | 布局龙头，等待出清 |
| 上行 | 需求回暖、涨价、产能紧张 | 重仓持有 |
| 顶部 | 全行业暴利、疯狂扩产 | 逐步减仓 |
| 下行 | 需求下滑、库存累积、价格下跌 | 回避或做空 |

**判断方法**：
- 产品价格走势（涨价=上行，跌价=下行）
- 产能利用率（>80%景气，<60%低迷）
- 厂商扩产节奏（疯狂扩产可能见顶）
- 库存水平（库存累积=下行信号）

## 二、公司定位分析（自下而上）

**核心问题**：这家公司在行业里排第几？靠什么赚钱？对上下游有没有议价权？

### 2.1 主营业务构成

| 分析要点 | 具体内容 |
|---------|---------|
| 各业务营收占比 | 哪个是主力、哪个是增长点 |
| 各业务毛利率 | 哪个赚钱、哪个不赚钱 |
| 业务相关性 | 是否协同、还是杂乱拼凑 |

**关键判断**：
- 主营业务要聚焦，太杂的公司往往不专业
- 毛利率低于5%的业务，基本是给上游打工
- 高毛利率业务占比提升，说明结构优化

### 2.2 市场地位

| 地位 | 特征 | 投资价值 |
|-----|------|---------|
| 全球第一 | 规模或技术领先 | 最高 |
| 国内第一 | 国产替代龙头 | 高 |
| 细分龙头 | 某一细分领域领先 | 中高 |
| 跟随者 | 规模偏小、技术跟随 | 中低 |

### 2.3 核心客户

| 维度 | 看什么 | 预警信号 |
|------|--------|----------|
| 客户集中度 | 前五大客户占比 | >50%依赖大客户 |
| 客户质量 | 是否进入头部供应链 | 英伟达、苹果认证=高壁垒 |
| 客户稳定性 | 合作年限、复购率 | 频繁更换客户=产品力不足 |

### 2.4 上下游议价能力

| 维度 | 看什么 | 强势信号 |
|------|--------|----------|
| 对上游 | 原材料占比、供应商数量 | 供应商分散、原材料占比低 |
| 对下游 | 客户集中度、预收款 | 客户分散、先收钱后发货 |
| 应收账款 | 周转天数、占营收比 | 周转快、占比低 |
| 存货 | 周转天数 | 周转快=产品好卖 |

**关键判断**：
- 预收账款高 = 产品抢手，客户先付钱
- 应收账款高 = 被客户占用资金，议价权弱
- 存货周转天数暴增 = 卖不动了

### 2.5 国际化程度

| 维度 | 看什么 | 意义 |
|------|--------|------|
| 海外收入占比 | 国际化程度 | >30%说明有国际竞争力 |
| 海外产能布局 | 是否有海外工厂 | 规避关税、贴近客户 |
| 国际认证 | 是否进入国际供应链 | 英伟达、苹果认证=高壁垒 |
| 地缘政治风险 | 制裁/关税风险 | 半导体、军工敏感 |

## 三、管理层分析

**核心问题**：管理层是否可信？利益是否绑定？过往承诺是否兑现？

### 3.1 管理层背景

| 维度 | 看什么 | 好的信号 |
|------|--------|----------|
| 专业背景 | 技术出身还是资本运作出身 | 技术出身更踏实 |
| 行业经验 | 在行业待了多少年 | 深耕多年 |
| 学历/职称 | 是否有技术积累 | 工程院院士、博士等 |

### 3.2 利益绑定

| 维度 | 看什么 | 好的信号 |
|------|--------|----------|
| 持股比例 | 实控人/管理层持股 | >10%利益绑定 |
| 薪酬水平 | 与行业对比 | 合理水平 |
| 股权激励 | 是否有、目标是否激进 | 有激励且目标明确 |
| 增减持 | 自家股票买卖 | 增持=看好，减持=套现 |

### 3.3 过往承诺兑现

| 维度 | 看什么 | 预警信号 |
|------|--------|----------|
| 业绩承诺 | 并购标的是否达标 | 连续不达标 |
| 募投项目 | 是否按计划推进 | 频繁延期、变更 |
| 分红承诺 | 是否按承诺分红 | 说好的不分红 |
| 战略规划 | 说到的是否做到 | 画饼不兑现 |

### 3.4 资本运作记录

| 行为 | 好的信号 | 坏的信号 |
|------|----------|----------|
| 定增 | 用于主业扩产 | 频繁定增、用途不明 |
| 并购 | 整合能力强、标的优质 | 高溢价并购、商誉大 |
| 减持 | 少量减持、有合理解释 | 大额减持、清仓式减持 |
| 质押 | 低质押率 | 高质押率（>80%） |

## 四、财务数据验证

**核心问题**：赚钱能力怎么样？增长是否可持续？财务是否健康？

### 4.1 盈利能力

| 指标 | 公式 | 参考值 | 意义 |
|------|------|--------|------|
| 毛利率 | (营收-成本)/营收 | 制造业>20% | 产品竞争力 |
| 净利率 | 净利润/营收 | >10% | 综合盈利能力 |
| ROE | 净利润/净资产 | >15% | 股东回报效率 |
| ROIC | EBIT/(有息负债+净资产) | >10% | 资本回报效率 |

### 4.1.1 PEG指标（成长股估值利器）

**核心问题**：PE看着高，但增速能不能消化？

| 指标 | 公式 | 意义 |
|------|------|------|
| PEG | PE / 净利润增速(%) | 增速调整后的估值 |

**判断标准**：

| PEG | 判断 | 说明 |
|-----|------|------|
| <0.5 | 严重低估 | 增速远超估值，罕见机会 |
| 0.5-1 | 合理偏低 | 增速能消化估值，成长股甜蜜区 |
| 1-1.5 | 合理 | 估值与增速匹配 |
| 1.5-2 | 偏高 | 需要超预期增速才能撑住 |
| >2 | 高估 | 增速跟不上估值，危险 |

**使用场景**：
- 适用于**高增速成长股**（净利润增速>20%）
- 不适用于周期股（增速波动大，PEG无意义）
- 不适用于亏损公司（没有正增速）
- **和PE互补**：PE看绝对贵不贵，PEG看相对值不值

**案例**：
- 天赐材料PE=50倍，Q1净利+80%，PEG=50/80=0.63 → 看着贵其实不贵
- 某公司PE=30倍，Q1净利+15%，PEG=30/15=2.0 → 看着便宜其实贵了

### 4.1.2 个股PE与行业PE对比（重要）

**核心问题**：当前估值是贵还是便宜？和同行比有没有溢价？

| 溢价率 | 判断 | 策略 |
|--------|------|------|
| <0% | 低估 | 可能有机会（但要查原因） |
| 0-30% | 合理 | 正常估值 |
| 30-80% | 偏高 | 需要增速或壁垒支撑 |
| 80-150% | 高估 | 谨慎，等待回调 |
| >150% | 严重高估 | 回避，风险极大 |

**计算公式**：
```
溢价率 = (个股PE - 行业平均PE) / 行业平均PE × 100%
```

**分析模板**：

| 公司 | 股票PE | 行业平均PE | 溢价率 | 估值判断 |
|------|--------|-----------|--------|----------|
| | | | | |

**判断逻辑**：
- **高增速可以消化高溢价**：如Q1净利+100%，PE高于行业平均50%仍可接受
- **高壁垒可以支撑高溢价**：如英伟达独家供应商，PE高于行业平均80%仍合理
- **低增速+高溢价=危险**：如Q1仅+14%，PE高于行业平均100%则严重高估
- **纯题材+高溢价=炒作**：如算力经销业务，PE高于行业平均300%则是纯炒作

**行业平均PE参考值（2026年）**：

| 行业 | 平均PE范围 | 说明 |
|------|-----------|------|
| 电子代工 | 20-25倍 | 毛利率低，估值偏低 |
| PCB | 30-40倍 | 技术壁垒中等 |
| 覆铜板 | 25-35倍 | 周期+成长双重属性 |
| 锂电材料 | 20-30倍 | 周期波动大 |
| 清洁能源 | 15-25倍 | 增速稳定但不高 |
| 半导体设备 | 40-60倍 | 高壁垒+高成长 |
| AI算力 | 30-50倍 | 高成长+高预期 |
| 消费电子 | 15-20倍 | 成熟行业 |

**毛利率试金石**：
- <5%：基本是给上游打工
- 5-15%：普通制造业
- 15-30%：有一定壁垒
- 30-50%：强技术或品牌溢价
- >50%：垄断或平台型公司

### 4.2 成长能力

| 指标 | 看什么 | 好的信号 |
|------|--------|----------|
| 营收增速 | 行业景气度 | >15% |
| 净利润增速 | 盈利弹性 | >营收增速（规模效应） |
| 扣非净利润增速 | 主业真实增长 | 与净利润增速匹配 |
| 经营现金流增速 | 赚的是真金白银 | 与净利润增速匹配 |

### 4.3 财务健康度

| 经营现金流 | 正 | 负=赚的是纸面利润 |
| 商誉/净资产 | <30% | >50%减值风险大 |
| 质押率 | <50% | >80%风险大 |

### 4.4 研发投入

| 指标 | 参考值 | 意义 |
|------|--------|------|
| 研发费用率 | 制造业>5% | 重视研发 |

| 发明专利数量 | 行业前列 | 技术积累 |
| 研发资本化率 | <30% | 过高可能是美化利润 |

### 4.5 关联交易与治理风险

# PART 3：分析技巧与常见陷阱

## 分析技巧

### 1. 先看风险，再看机会

对烂的东西毫不留情——基本面烂透了就直接说，别给人画饼。

### 2. 用数据说话，别用感觉

"光通信占比3.73%"——这就是数据，不是感觉。

### 3. 关注矛盾点

业绩巨亏，股价却能连板——找出矛盾，才能理解市场在炒什么。

### 4. 股东户数是散户情绪温度计

户数暴增 = 散户蜂拥而入 = 危险
户数持续减少 = 筹码集中 = 可能有机会

### 5. 毛利率是试金石

- <5%：给上游打工
- 5-15%：普通制造业
- 15-30%：有一定壁垒
- 30-50%：强技术或品牌溢价
- >50%：垄断或平台型公司

### 6. 产业链验证

公司说自己好不算好，要从上下游数据交叉验证。

---

### 6. 对比公司时必须明确业务范围

**Pitfall（用户纠正过）**：用户会问"你对比的是电缆业务还是光纤业务？"——当对比多家公司时，必须先明确对比的是哪个业务板块，因为同一行业的公司业务结构可能差异很大。

**做法**：
- 先列出各家公司的业务结构对比表
- 明确标注"本次对比主要围绕XX业务"
- 如果业务重叠度低，要说明哪些公司在这个业务上直接竞争、哪些不竞争

## 飞书消息格式规范（重要）

**核心规则**：飞书文本消息不支持markdown表格渲染，会显示原始语法。**必须使用卡片消息+原生table组件展示表格数据，不要偷懒用纯文本。**

### 格式选择

| 场景 | 推荐格式 | 说明 |
|------|----------|------|
| 有表格数据 | 卡片消息（interactive）+ 原生table组件 | **首选方案**，支持表格渲染 |
| 简单列表 | 纯文本 + emoji + 分隔线 | 降级方案，用✓/✗做标记 |
| 需要加粗 | 富文本（post） | 支持加粗、链接等样式 |

**Pitfall（用户明确纠正过）**：用户会说"你现在发消息直接不使用表格了是么"——说明默认必须用表格，纯文本是偷懒行为。每次发送带数据对比的内容时，必须用卡片消息。

**Pitfall（2026-06-22用户纠正）**：用户明确要求"尽可能不要使用md表格"。这个偏好适用于所有平台的回复（不仅是飞书）。在普通对话中展示数据对比时，用纯文本列表、分条列示、emoji标记等方式，不要用markdown表格语法。飞书卡片+原生table组件仍然是首选（因为飞书支持渲染），但md表格在任何场景下都要避免。

### Lark-MCP调用稳定性（重要）

**Pitfall**：连续快速调用 `mcp_lark_mcp_im_v1_message_create` 会导致lark-mcp服务器崩溃（"MCP server is unreachable after N consecutive failures"）。

**原因**：卡片消息的content字段包含大量JSON嵌套，每次调用负载较大。连续3-5次快速调用会打崩服务器。

**解决方案**：
1. 卡片消息内容尽量精简，避免过长的JSON
2. 如果连续调用失败，等待30-60秒服务器自动恢复
3. 如果持续失败，提示用户手动执行 `hermes mcp restart lark-mcp`
4. 可以先用纯文本在对话中回复摘要，等服务器恢复后再发卡片
