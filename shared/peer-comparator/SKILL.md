# 可比公司对比分析框架

配合 `stock-analysis-base` 使用。本skill专注**多公司横向对比**，找出相对优劣。

## 核心原则

**Pitfall（用户纠正过）**：对比公司时必须先明确对比的是哪个业务板块。同一行业的公司业务结构可能差异很大（比如亨通光电有电缆+光纤+海缆，对比时必须说清楚比的是哪块业务）。

---

## 一、对比前准备

### 1.1 业务结构摸底

**在做任何对比之前，先搞清楚每家公司的业务结构。**

| 公司 | 业务1 | 业务2 | 业务3 | 主营占比 |
|------|-------|-------|-------|----------|
| 公司A | XX（营收占比XX%） | XX（XX%） | XX（XX%） | |
| 公司B | XX（营收占比XX%） | XX（XX%） | XX（XX%） | |
| 公司C | XX（营收占比XX%） | XX（XX%） | XX（XX%） | |

**关键问题**：
- 这几家公司到底在哪个业务上直接竞争？
- 业务重叠度高不高？
- 本次对比主要围绕哪个业务？

### 1.2 确定对比范围

明确声明："本次对比主要围绕XX业务板块"，然后标注每家公司在该业务上的营收占比。

## 二、多维对比框架

### 2.1 规模对比

| 公司 | 营收 | 净利润 | 市值 | 员工数 |
|------|------|--------|------|--------|
| | | | | |

### 2.2 盈利能力对比

| 公司 | 毛利率 | 净利率 | ROE | ROIC |
|------|--------|--------|-----|------|
| | | | | |

**判断标准**：
- 毛利率差距>10个百分点 = 产品力或成本结构有本质差异
- ROE差距大 = 资本效率差异，可能和杠杆有关（结合资产负债率看）

### 2.3 成长能力对比

| 公司 | 营收增速 | 净利润增速 | 扣非净利增速 | 经营现金流增速 |
|------|----------|------------|-------------|---------------|
| | | | | |

**关键**：不仅看增速绝对值，还要看**趋势**——是在加速还是在减速。

### 2.4 估值对比

| 公司 | PE | PB | PEG | PE溢价率(相对行业) |
|------|-----|-----|-----|-------------------|
| | | | | |

**PE溢价率计算**：
```
溢价率 = (个股PE - 行业平均PE) / 行业平均PE × 100%
```

**判断逻辑**：
- 高增速+低PEG = 性价比高
- 低增速+高PE溢价 = 危险
- 高壁垒+高PE溢价 = 可以接受

### 2.5 财务健康对比

|------|-----------|---------|-----------|--------|
| | | | | |

### 2.6 股东博弈对比

## 三、PE与行业平均PE深度对比

**这是对比分析的核心工具。**

### 3.1 对比模板

| 公司 | 股票PE | 行业平均PE | 溢价率 | 增速 | PEG | 估值判断 |
|------|--------|-----------|--------|------|-----|----------|
| | | | | | | |

### 3.2 行业平均PE参考值（2026年）

| 行业 | 平均PE范围 | 说明 |
|------|-----------|------|
| 电子代工 | 20-25倍 | 毛利率低，估值偏低 |
| PCB | 30-40倍 | 技术壁垒中等 |
| 覆铜板 | 25-35倍 | 周期+成长双重属性 |
| 锂电材料 | 20-30倍 | 周期波动大 |
| 清洁能源 | 15-25倍 | 增速稳定但不高 |
| 半导体设备 | 40-60倍 | 高壁垒+高成长 |
| AI算力 | 30-50倍 | 高成长+高预期 |
| 消费电子 | 15-20倍 | 成熟行业 |

### 3.3 综合判断矩阵

```
                    低PE溢价          高PE溢价
高增速（>30%）    → 最佳标的         → 需要壁垒支撑
低增速（<15%）    → 价值陷阱风险     → 严重高估，回避
```

## 五、常见对比陷阱

| 陷阱 | 表现 | 如何避免 |
|------|------|----------|
| 业务不对齐 | 拿电缆公司和光纤公司比 | 先列业务结构再对比 |
| 规模不对齐 | 拿千亿市值和百亿市值比PE | 同体量公司对比更有意义 |
| 周期位置不对齐 | 上行期龙头 vs 底部期龙头 | 标注各自周期位置 |
| 忽略PEG | 只看PE说贵不贵 | 高增速公司必须算PEG |
| 忽略股东博弈 | 只看财务数据 | 户数暴增的公司财务再好也要警惕 |
