# 算电协同行业分析（2026-06-23）

## 行业定义

算电协同 = 算力网与电力网深度融合、双向赋能。核心是"电支撑算、算优化电"。

2026年首次写入政府工作报告，与"东数西算"并列成为国家级新基建。

## 核心逻辑

1. **需求侧**：AI算力爆发，数据中心用电量年均增速超12%，2030年将突破945太瓦时
2. **供给侧**：西部风光电丰富但消纳难，算力中心成为绿电"超级大买家"
3. **政策侧**：新建智算中心绿电占比要达80%以上

## 产业链四大环节

### 1. 绿电运营商（卖电的）
- 华润新能源（001248）：装机4158万千瓦，即将登陆深主板
- 龙源电力（00916.HK）：装机43.2GW，风电龙头
- 三峡能源：水电+新能源双轮驱动
- 华电新能：火电协同，消纳能力强
- 大唐新能源（01798.HK）：估值便宜，PB仅0.5倍
- **协鑫能科（002015）**：绿电+算力双轮驱动，算电协同概念最纯正

### 2. 智算中心/IDC（卖算力的）
- 利通电子（603629）：从金属结构件转型算力租赁
- 注意：很多公司靠"租金差"赚钱，毛利率不高

### 3. 液冷与温控（降温的）
- 液冷（浸没式）将成为标配
- 余热回收还能供暖

### 4. 电网设备（铺路的）
- 特高压、智能电网、配电网升级

## 投资逻辑

- **绿电运营商**：最稳，赚稳定现金流+估值修复
- **液冷储能**：技术弹性大
- **算力调度平台**：高风险高回报

## 关键数据

- 全球数据中心用电增速是全球平均用电增速的4倍以上
- 电费占数据中心运营成本的60%-70%
- 西部新能源电价约0.2-0.3元/度，东部0.6-0.8元/度
- 算力中心绿电占比要求80%以上

## 风险提示

1. 政策风险：绿电占比要求执行力度待观察
2. 技术风险：液冷、储能技术还不成熟
3. 竞争风险：算力租赁赛道已拥挤
4. 周期风险：算力需求波动大，电力投资周期长
