# 半导体材料国产替代估值基准（2026年6月）

## 板块整体估值水平

半导体材料国产替代板块估值偏贵，核心逻辑是"国产化率从X%到Y%能抢多少份额"，而非传统PE定价。

**板块平均PE（TTM）**：60-100倍
**按2026年预测利润算（Forward PE）**：35-60倍

## 估值定价公式

```
合理PE = 基础PE（30-40倍）+ 国产替代溢价 + 技术壁垒溢价 + 业绩弹性溢价
```

- 国产化率越低（<10%），溢价越高 → 光刻胶、前驱体PE最贵
- 国产化率越高（>30%），溢价越低 → CMP抛光液、靶材PE相对合理
- 业绩已有兑现的，PE更实在 → 安集、雅克、江丰
- 业绩未兑现、纯靠预期的，PE虚高 → 江化微、沪硅产业

## 分梯队估值参考

### 第一梯队：纯正龙头，业绩已兑现（PE 40-60倍）

- **安集科技**（CMP抛光液）：PE 50-60倍，国产化率30%+，利润5亿+
- **雅克科技**（前驱体+电子特气）：PE 40-50倍，前驱体进入国际大厂
- **江丰电子**（靶材）：PE 50-60倍，全球靶材市占率第二
- **华特气体**（电子特气）：PE 50-60倍，品种丰富
- **鼎龙股份**（CMP抛光垫）：PE 50-60倍，国内唯一

### 第二梯队：高成长+高预期（PE 60-100倍）

- **中巨芯**（湿电子化学品+电子特气）：PE 80-100倍，G5放量
- **晶瑞电材**（试剂+光刻胶）：PE 60-80倍，KrF量产
- **南大光电**（MO源+ArF光刻胶）：PE 80-100倍，ArF唯一量产
- **彤程新材**（光刻胶）：PE 80-100倍，北京科华KrF

### 第三梯队：概念最纯但业绩最薄（PE 100倍+）

- **江化微**（湿电子化学品）：PE 170倍，营收增速12%、净利增速6%
- **沪硅产业**（大硅片）：利润极薄或亏损，PE失真

## 关键结论

- 同样花1块钱买利润，安集科技要花50块，江化微要花170块
- 除非江化微未来利润增速是安集的3倍以上，否则170倍PE撑不住
- 选股优先级：业绩兑现度 > 国产替代逻辑 > 概念纯度

## 湿电子化学品行业补充

- 全球市场由欧美日主导（欧美32%+日本28%+亚洲其他38%）
- 国内企业约40家，龙头市占率仅5%左右
- 国产化率：集成电路用约35%，显示面板用约40%
- G5等级（先进制程）是最高壁垒，国内能做的企业很少
- 部分通用品种已出现结构性过剩（电子级硫酸、氨水等）
